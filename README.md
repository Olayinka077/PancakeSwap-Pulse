# PancakeSwap-Pulse

PancakeSwap Pulse: User Growth, Swap Trends & Token Activity on BNB Chain
Project Overview

This project provides a comprehensive on-chain analysis of PancakeSwap’s performance on the BNB Smart Chain, leveraging data from the dex.trades table on Dune Analytics. The goal is to understand user behavior, trading trends, and token activity over time.

Dashboard Dimensions

The analysis explores three major dimensions of PancakeSwap’s ecosystem:

User Dynamics

Daily active traders

Cumulative user growth

Retention patterns (new vs. returning traders)

Trading Activity

Swap volume trends

Average swap size

Liquidity contribution from new users

Market Insights

Top traded token pairs by USD volume

Key Insights

Steady user growth with consistent participation from both new and returning traders.

Trading activity dominated by BNB and stablecoin pairs, indicating strong liquidity and trust.

Periodic spikes in new user volume correlate with market movements and project milestones.

Impact

These insights provide actionable intelligence for:

Developers optimizing user experience and ecosystem features

Liquidity providers managing risk and rewards

DeFi researchers analyzing market demand, retention, and trading patterns
